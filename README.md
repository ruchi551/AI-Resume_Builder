# AI Resume Builder

AI Resume Builder is a full-stack web application that allows users to create professional and ATS-friendly resumes easily. The platform provides real-time resume preview, customizable templates, and AI-powered suggestions to help users write better resume content.

The project is built using the MERN Stack (MongoDB, Express.js, React.js, Node.js) and focuses on providing a clean UI and smooth resume creation experience.

---


## ✨ Features

- Create professional resumes quickly  
- Real-time resume preview while editing  
- Multiple resume templates  
- AI-powered content suggestions  

### Resume Sections
- Personal Information  
- Education  
- Experience  
- Skills  
- Projects  

### Additional Features
- Secure user authentication  
- Download resume as PDF  
- Responsive design for all devices

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
git clone https://github.com/ruchi551/Resume_Builder.git 


### 2️⃣ Navigate to project folder
cd Resume_Builder


### 3️⃣ Install dependencies

#### Client
cd client
npm install


#### Server


cd ../server
npm install


---

## ▶️ Run the Application

### Start backend server


cd server
npm run dev


### Start frontend


cd client
npm run dev


Open in browser:


http://localhost:5173


---

## 🔑 Environment Variables

Create a `.env` file inside the **server** folder.
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
IMAGEKIT_PUBLIC_KEY=your_key
IMAGEKIT_PRIVATE_KEY=your_key
IMAGEKIT_URL_ENDPOINT=your_endpoint
